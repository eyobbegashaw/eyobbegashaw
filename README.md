## ⚡ Pro Tips (This is what actually makes it “senior level”)

- Don’t overload with GIFs → use **section-based animation**
- Keep **dark + neon theme consistent**
- Use **motion to guide attention**, not distract
- Combine:
  - `capsule-render` (headers)
  - `skillicons`
  - `giphy (3D-like loops)`
  - `gradients`
